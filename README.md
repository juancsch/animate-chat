# animate-chat

Un chat en tiempo real usando gif animados como videos

# Tecnologías usadas

Node y NPM

Cliente:
* webpack
* socketio
* handlebars
...

Servidor:
* express
* socketio
* leveldb
* ffmpeg: programa externo
...

Testing:
* chimp: mocha, webdriverio, ...
* travis-ci
