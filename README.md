# animate-chat

Un chat en tiempo real usando gif animados

# Tecnologías usadas

Cliente: basada en una arquitectura hexagonal
* webpack
* babel
* eslint
* socketio
* handlebars
* webrtc
...

Servidor: basada en una arquitectura hexagonal
* typescript
* tslint
* express
* socketio
* leveldb
* ffmpeg
...

Testing, E2E y CI:
* jest
* webdriverio
* travis-ci

# Despliegue en cloud

Heroku: [animate-chat](https://animate-chat.herokuapp.com/)
